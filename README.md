# Visual File 可视化文件

## Features

以图形化的方式显示你的项目结构

Display your project structure in a graphical way.

## Usage

按 <kbd>F1</kbd> 打开命令面板，输入 `Visual File` 打开可视化文件。

Press <kbd>F1</kbd> to open the command palette, type `Visual File` to open the visual file.
